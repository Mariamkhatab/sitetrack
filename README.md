# SiteTrack — Site Issue & Snagging Tracker

A professional WPF desktop app for QA/QC engineers in construction to log defects, assign responsibilities, and export comprehensive reports.

## 🌐 Website

The marketing site is hosted on GitHub Pages at: **https://Mariamkhatab.github.io/sitetrack/**

## 🚀 Quick Start

### Deployment (GitHub Pages)

This site is automatically deployed via GitHub Pages from the `main` branch.

**To enable/verify GitHub Pages:**
1. Go to your repository settings: https://github.com/Mariamkhatab/sitetrack/settings
2. Scroll to **Pages** section
3. Under "Build and deployment":
   - Set **Source** to: `Deploy from a branch`
   - Set **Branch** to: `main` and folder to `/root`
   - Save

Your site will be live at: `https://Mariamkhatab.github.io/sitetrack/`

### Local Development

Simply open `index.html` in your browser to preview the site locally.

- Full issue logging for construction snags
- Priority and status management
- Excel export with multi-sheet reports
- Local JSON data storage (offline-first)
- Zero cloud accounts required
- Built with C# .NET 8 WPF
